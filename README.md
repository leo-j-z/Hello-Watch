# Hello Watch 项目说明：
该项目是一个基于STM32F4芯片，FreeRTOS系统，LVGL图形库开发的穿戴式嵌入式项目，主要展示本人对于FreeRTOS和Cortext-M4内核MCU应用的理解，该项目集成了
- 日期时间显示
- 温湿度监测
- 步数记录
- 电量显示
- 心率检测
- 抬腕亮屏，自动息屏
- 亮度调节
- BLE蓝牙数据传输
- 日历
- 计算器
- 秒表计时
- 腕上游戏

等多种功能，
***
Master分支下是本项目的工程文件，其中Hardeware目录下是本项目的硬件原理图，Hello_Watch目录下是项目的软件代码部分，images目录下是实物图的功能演示部分
